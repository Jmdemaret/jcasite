Connectez-vous pour modifier le site

Cette interface permet de modifier le contenu du site (messages, photos, actualités, avis…) sans toucher au design. Une seule connexion initiale est nécessaire.

Comment obtenir ma clé d'accès ?
  1. Cliquez sur ce lien pour créer une clé (ouverture sur GitHub, page pré-remplie).
  2. Si demandé, connectez-vous avec le compte GitHub du club.
  3. Durée : choisissez 1 an (ou « No expiration » si disponible).
  4. Scope déjà coché : public_repo — ne modifiez rien.
  5. Cliquez « Generate token » tout en bas.
  6. Copiez la clé qui s'affiche (elle commence par ghp_…).
  7. Revenez ici, collez-la dans le champ ci-dessous et cliquez Se connecter.

La clé reste dans ce navigateur uniquement — elle n'est jamais envoyée ailleurs que vers GitHub.
